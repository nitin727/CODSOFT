# TASK 5 - Student Management System (Java)

## 📌 Description
This project is a simple **Student Management System** developed in **Java** as part of the **CodSoft Java Development Internship**.

The program allows users to add, view, and remove student records using a simple console-based menu system.

## 🚀 Features
• Add new student  
• View all students  
• Remove student  
• Menu driven program  
• Uses ArrayList to store student records  

## 🛠 Technologies Used
Java  
VS Code  
GitHub  

## ▶️ How to Run

Compile the program

javac StudentManagementSystem.java

Run the program

java StudentManagementSystem
