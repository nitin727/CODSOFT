# Task 4 – Currency Converter

This project is a simple **Currency Converter** developed using **Java** as part of the **Java Development Internship at CodSoft**.

## 💡 Description
The program allows users to convert currencies between **INR, USD, and EUR** using predefined conversion rates.

## ⚙️ Features
- Convert INR to USD and EUR
- Convert USD to INR and EUR
- Simple console-based interface
- User input handling with Scanner

## 🛠️ Technologies Used
- Java
- VS Code
- GitHub
